## :arrow_down: Install

- Install with [yarn](https://yarnpkg.com/)

```bash
$ yarn add @commitlint/cli @lucasbernardolira/custom-commitlint-config -D

$ echo "module.exports = {extends: ['@lucasbernardolira/custom-commitlint-config']};" > commitlint.config.js
```
